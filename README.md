# 本科破局系统 — 开发说明书

## 1. 项目简介

> 这个项目是给中国的本科生进行使用，目的是利用AI agent帮助大家完成本科期间一系列冗杂的任务，比如写课程作业（用AI 写latex完成），完成大作业报告（PPT 也使用latex完成），完成调研任务等等。 初期开发我们只做前两个功能 ：课程作业和大作业报告，都基于latex

## 2. 核心目标

> 这个系统最终要达到什么效果？用户用完之后，生活/学习会有什么不同？
> 这个系统最终会使中国的本科生极大的释放双手，在短短十分钟内就可以完成以前需要耗费一下午的任务，并且完成的更好更精致。用户使用后会感觉到生产力极大的解放，从而有更多的事件完成自己的事情

## 3. 用户画像

> 目标用户是谁？他们现在的痛点是什么
> 中国的本科生 代表日常生活中需要处理很多冗杂而且无用的事情的人群

## 4. 功能模块

> 列出所有你想做的功能，每个模块一行，后面简单说它解决什么问题。
> 整个代码是一个网页应用的形式，计算（大模型/latex编译等）均在云端服务器进行。
> 大致的工作结构如下 ：
> 用户 -> 新建一个课程/项目 -> 课程分为几个部分 ：
> 1. 公共区 ： 上传公共资料（讲义，课本，练习题），这些作为AI的一个上下文
> 2. 任务区 ： 用户撰写本次的任务。 可以上传文件，或者书写文字
> 一个例子 ： 用户新建一门课程叫做回归分析 在公共区上传了回归分析中的课本（可能这个文件很大）和老师的讲义。然后在任务区新建了一个任务，上传了第一次pdf作业。 

- [ ] 模块一：作业系统/任务管理/公共区的材料预处理
- [ ] 模块二：任务发送/任务接受
- [ ] 模块三：服务器端配套代码

> 用户从打开软件到完成任务，经历哪些步骤？把你自己的工作流写在这里，这就是产品的核心逻辑。
